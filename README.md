# tag-cloud
Tag Cloud Implemetation

This folder constists of two folder: client and server

# Client
This is a Vite + React + Vite project

### Installation
```bash
pnpm install
pnpm run dev
```

# Client
This is a NestJS project that serves as the main api service for the tag cloud

### Installation
```bash
pnpm install
pnpm run start:dev
```
